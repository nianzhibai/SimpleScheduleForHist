# 课表App - Material Design 3 设计指南

## 设计理念

### 1. 视觉层次
```
┌─────────────────────────────────────┐
│  天气概览卡片 (Weather Hero Card)   │  ← 第一层级: 用户打开App首先看到
├─────────────────────────────────────┤
│  今日课表 (Timeline Course Cards)   │  ← 第二层级: 核心功能
├─────────────────────────────────────┤
│  快捷操作 (Quick Actions)           │  ← 第三层级: 辅助功能
└─────────────────────────────────────┘
```

### 2. 颜色系统
```
主色 Primary: #6366F1 (Indigo)
辅色 Secondary: #14B8A6 (Teal)
成功色: #10B981 (Emerald)
警告色: #F59E0B (Amber)
错误色: #EF4444 (Red)

深色模式:
- 背景: #111827 (Gray900)
- 卡片: #1F2937 (Gray800)
- 文字: #F9FAFB (Gray50)
```

### 3. 间距系统 (8dp网格)
```
xs: 4dp   ← 紧凑元素
sm: 8dp   ← 元素内部
md: 16dp  ← 组件间距
lg: 24dp  ← 区块间距
xl: 32dp  ← 页面边距
xxl: 48dp ← 大区块
```

### 4. 圆角系统
```
xs: 4dp   ← 小标签
sm: 8dp   ← 按钮、输入框
md: 12dp  ← 小卡片
lg: 16dp  ← 主卡片
xl: 24dp  ← 大卡片
full: 9999dp ← 圆形
```

## 动效设计

### 1. 页面转场
```kotlin
// 淡入淡出 + 轻微上移
val enterTransition = fadeIn() + slideInVertically()
val exitTransition = fadeOut() + slideOutVertically()
```

### 2. 卡片交互
```kotlin
// 按压时缩小
scale: 0.98f (按下) → 1.0f (松开)

// 卡片进入
slideInVertically(offsetY = { it }) + fadeIn()
```

### 3. 天气图标动画
```kotlin
// 脉冲动画
pulse: 0.9f ↔ 1.1f, 周期2秒
```

### 4. 骨架屏
```kotlin
// Shimmer效果
translateX: 0f → 1000f → 0f
alpha: 0.3f ↔ 0.7f
```

## 深色模式适配

### 原则
1. 减少纯白/纯黑使用，改用Gray色阶
2. 降低对比度避免视觉疲劳
3. 减少大面积纯色，改用渐变
4. 天气卡片渐变色使用深色变体

### 颜色映射
```
浅色 → 深色
White → Gray800 (#1F2937)
Gray50 → Gray900 (#111827)
Primary50 → Primary900 (#312E81)
```
